# Python-Django-Pandas-Simple-API

## Installation

1. Install Flask Library
```
pip install flask
```

2. Install Pandas Library
```
pip install pandas
```

## Run Aplication

```
python app.py
```

### Postman testing

1. GET Method

```
curl --location 'http://localhost:5000/api/csv'
```

```
curl --location 'http://localhost:5000/api/newsapi'
```
